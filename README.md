# 🌐 Responsive Multi-Page Website

A modern, responsive multi-page website developed using **HTML5**, **CSS3**, and **JavaScript**. This project was completed as **Task 7** during my **Web Development Internship at Synent Technologies**.

The website demonstrates best practices in front-end development, including responsive layouts, reusable components, intuitive navigation, and clean code organization. It is designed to provide a seamless user experience across desktop, tablet, and mobile devices.
